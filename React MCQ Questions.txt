
---

1. What is React?

A. Database
B. JavaScript Library
C. Programming Language
D. Framework

Answer: B
Explanation: React is a JavaScript library for building UI.

---

2. React was developed by:

A. Google
B. Microsoft
C. Facebook
D. Amazon

Answer: C
Explanation: React was developed by Facebook (Meta).

---

3. What is JSX?

A. Java Syntax Extension
B. JavaScript XML
C. JSON XML
D. JavaScript Extension

Answer: B
Explanation: JSX stands for JavaScript XML and allows writing HTML inside JavaScript.

---

4. Which method is used to render React content?

A. ReactDOM.create()
B. React.render()
C. ReactDOM.render()
D. renderDOM()

Answer: C
Explanation: ReactDOM.render() renders content into DOM (React 17 and earlier).

---

5. What is a React Component?

A. HTML File
B. CSS File
C. Reusable UI Block
D. Database Table

Answer: C
Explanation: Components are reusable UI building blocks.

---

6. Which of the following is a Functional Component?

A.

```js
function App() {
  return <h1>Hello</h1>;
}
```

B.

```js
class App {}
```

C. Both
D. None

Answer: A
Explanation: Functional component is a JavaScript function returning JSX.

---

7. Props are:

A. Internal state
B. Read-only inputs to components
C. Database values
D. Hooks

Answer: B
Explanation: Props are read-only data passed from parent to child.

---

8. State in React is:

A. External variable
B. Local component data
C. CSS property
D. API

Answer: B
Explanation: State stores dynamic data inside a component.

10. Which Hook is used for side effects?

11. What does useEffect run by default?

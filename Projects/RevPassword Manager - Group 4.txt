Application Overview

The Password Manager is a secure console-based application that allows users to safely store and manage passwords for their various online accounts. Users can create an account with a master password, log in, generate strong random passwords, store account credentials, and manage their password vault. The application emphasizes security through password encryption and master password protection.

Core Functional Requirements

As a User, I should be able to:

Create an account to securely manage my passwords
Log in to my account to access the application
Update my profile information like name and email
Generate strong random passwords by providing parameters like string length, characters, numbers, etc.
List all the passwords with the respective account names
View individual password details (with master password re-entry)
Add new password entries for various accounts
Update existing passwords
Delete password entries
Search passwords by account name
Add security questions for account recovery
Get a verification code for sensitive operations which expires after use
Manage security questions linked to my account
Standard Functional Scope

Registered users should be able to log in, change the password and request for a forgotten password (recovered through security questions).

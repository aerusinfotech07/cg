# Node.js Notes

## What It Is
- Node.js is a free, open source tool that runs JavaScript outside the browser.
- It is used to build fast, scalable apps like web servers, APIs, and tools.

## Why It Scales
- Event-driven, non-blocking model.
- Handles many connections at once without waiting for each to finish.
- Good for real-time apps and high-traffic websites.

## What You Can Build
- Web servers and websites
- REST APIs
- Real-time apps (chat)
- Command-line tools
- File and database tools
- IoT and hardware control

## How To Run Code
1. Save code in a file (e.g., app.js).
2. Run in terminal: `node app.js`.

## Learning By Examples
- "Show Node.js" shows both code and result.
- Some examples display output in a CLI screen.

## Example: Simple Server
- Use `http` module.
- Responds with `Hello World!` on port 8080.

## Example: CLI Output
- `console.log` prints to the command line.

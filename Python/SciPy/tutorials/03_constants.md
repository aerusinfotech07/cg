# SciPy Constants

## Physical Constants

```python
from scipy import constants

print(constants.pi)
print(constants.c)
print(constants.h)
print(constants.e)
print(constants.G)
print(constants.g)
```

## Mathematical Constants

```python
from scipy import constants
print(constants.golden)
```

## Common Constants

| Constant | Description |
|----------|-------------|
| pi | Pi |
| c | Speed of light |
| h | Planck constant |
| e | Elementary charge |
| G | Gravitational constant |
| g | Standard gravity |
| golden | Golden ratio |
| Avogadro | Avogadro constant |

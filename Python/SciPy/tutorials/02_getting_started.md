# SciPy Getting Started

## Installation

```bash
pip install scipy
```

## Import

```python
import scipy
from scipy import optimize, constants, stats
```

## Version Check

```python
import scipy
print(scipy.__version__)
```

## Quick Example

```python
from scipy import constants
print("Pi:", constants.pi)
print("Speed of light:", constants.c)
```

## Dependencies

SciPy requires **NumPy**. Install both:

```bash
pip install numpy scipy
```

# SciPy Tutorials

SciPy is a scientific computing library for Python, built on NumPy. This folder contains topic-wise tutorials with runnable examples.

## Installation

```bash
pip install scipy
```

## Structure

```
SciPy/
├── README.md
├── TUTORIAL_INDEX.md
├── tutorials/          # Markdown tutorials
└── examples/           # Runnable Python examples
```

## How to Run Examples

From the `Python` folder:

```bash
python SciPy/examples/01_intro.py
```

## Topics

- Intro, Getting Started, Constants
- Optimizers, Sparse Data, Graphs
- Spatial Data, Matlab Arrays
- Interpolation, Significance Tests

See [TUTORIAL_INDEX.md](TUTORIAL_INDEX.md) for the full syllabus.

# NumPy Tutorials

NumPy (Numerical Python) is the foundation for scientific computing in Python. This folder contains topic-wise tutorials with runnable examples.

## Installation

```bash
pip install numpy
```

## Structure

```
NumPy/
├── README.md
├── TUTORIAL_INDEX.md
├── tutorials/          # Markdown tutorials
└── examples/           # Runnable Python examples
```

## How to Run Examples

From the `Python` folder:

```bash
python NumPy/examples/01_intro.py
```

## Topics

- Introduction, Getting Started, Creating Arrays
- Array Indexing, Slicing, Data Types
- Copy vs View, Shape, Reshape
- Iterating, Join, Split, Search, Sort, Filter

See [TUTORIAL_INDEX.md](TUTORIAL_INDEX.md) for the full syllabus.

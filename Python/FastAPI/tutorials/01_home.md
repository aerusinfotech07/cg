# FastAPI - Home

## Overview

**FastAPI** is a modern, high-performance web framework for building APIs with Python 3.8+, based on standard Python type hints.

## Key Features

- **Fast** – Very high performance (on par with NodeJS and Go)
- **Fast to code** – Increase development speed
- **Fewer bugs** – Reduce human errors via type hints
- **Automatic docs** – OpenAPI (Swagger) and ReDoc
- **Standards-based** – OpenAPI, JSON Schema

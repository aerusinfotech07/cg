# Pandas Tutorials

Pandas is the leading data analysis library for Python. This folder contains topic-wise tutorials with runnable examples.

## Installation

```bash
pip install pandas
```

## Structure

```
Pandas/
├── README.md
├── TUTORIAL_INDEX.md
├── tutorials/          # Markdown tutorials
└── examples/           # Runnable Python examples
```

## How to Run Examples

From the `Python` folder:

```bash
python Pandas/examples/01_intro.py
```

## Topics

- Intro, Getting Started, Series, DataFrames
- Read CSV, Read JSON, Analyzing Data
- Cleaning Data (Empty Cells, Wrong Format, Wrong Data, Duplicates)
- Correlations, Plotting

See [TUTORIAL_INDEX.md](TUTORIAL_INDEX.md) for the full syllabus.

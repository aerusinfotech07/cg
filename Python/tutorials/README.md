# Python Tutorials

A comprehensive topic-wise Python learning guide with tutorials, examples, and assignments.

## Structure

```
tutorials/
├── 01_introduction.md ... 50_file_write.md   # 50 tutorials
├── TUTORIAL_INDEX.md                         # Full syllabus & links
├── assignments/                              # Practice exercises
│   ├── 01_intro_assignment.md
│   ├── 02_getting_started_assignment.md
│   └── ... (50 assignments)
├── examples/                                  # Runnable Python examples
│   ├── 01_hello_world.py
│   └── ... (21+ examples)
└── README.md
```

## How to Use

1. **Study** – Read the tutorial (e.g., `06_output.md`)
2. **Practice** – Run examples from `examples/` folder
3. **Assign** – Complete exercises in `assignments/` (e.g., `06_output_assignment.md`)

## How to Run Examples

From the `Python` folder:

```bash
python tutorials/examples/01_hello_world.py
```

## Related Folders

- **NumPy/** – NumPy tutorials (arrays, indexing, slicing, etc.)
- **Pandas/** – Pandas tutorials (Series, DataFrames, CSV, cleaning, plotting)
- **SciPy/** – SciPy tutorials (constants, optimizers, sparse, graphs, spatial, interpolation, stats)
- **FastAPI/** – FastAPI tutorials (REST, params, Pydantic, CRUD, DB, WebSockets, deployment)
- **RestAPI-Axios/** – Python REST API (FastAPI) + frontend with Axios (JavaScript)

## Topics Covered

- **Core:** Intro, Get Started, Syntax, Statements, Variables, Output, Comments
- **Variables:** Names, Multiple Assignment, Output, Global
- **Data:** Types, Numbers, Casting, Strings, Booleans, Operators
- **Collections:** Lists, Tuples, Sets, Dictionaries
- **Control:** If/Else, Match, While, For Loops
- **Functions:** def, Range, Arrays, Iterators, Modules
- **Advanced:** datetime, math, JSON, RegEx, Try/Except, String Formatting, None, User Input
- **OOP:** Classes, Inheritance, Polymorphism, Encapsulation, Constructors, Methods, Properties
- **Files:** Read, Append, Operations, Paths, Write

See **[TUTORIAL_INDEX.md](TUTORIAL_INDEX.md)** for the complete syllabus with links.

## Requirements

# Assignment: Variable Names

Practice creating valid and well-named variables.

---

## Exercise 1 (Easy)
Create a variable named `student_count` and assign it the value 25. Print it.

**Expected Output:**
```
25
```

---

## Exercise 2 (Easy)
Which of these are valid variable names? Identify the invalid ones and explain why:
- `my_var`
- `2nd_place`
- `user_name`
- `class`
- `total_price`

---

## Exercise 3 (Easy)
Create variables using snake_case: `first_name`, `last_name`, and `age` with values "Alice", "Smith", and 30. Print them.

---

## Exercise 4 (Medium)
Create a constant (use UPPERCASE) named `MAX_LOGIN_ATTEMPTS` with value 5. Print it.

---

## Exercise 5 (Medium)
Fix the invalid variable names in this code (choose valid alternatives):
```python
# my-var = 10
# 2name = "test"
# for = 5
```

---

## Exercise 6 (Medium)
Write a line of code that would cause an error due to using a reserved keyword. Then write the corrected version using a valid name.

---

## Exercise 7 (Medium)
Improve these variable names to be more descriptive:
- `x` (stores a person's age)
- `n` (stores number of items in cart)
- `temp` (stores Celsius temperature)

---

## Exercise 8 (Hard)
Create variables for a product: `product_name`, `unit_price`, `quantity_in_stock`, and `is_available`. Use appropriate values and print them.

---

## Exercise 9 (Hard)
Which naming convention (snake_case, camelCase, PascalCase) should be used for each in Python?
- Variable: `user_count`
- Function: `get_total`
- Constant: `MAX_SIZE`
- Class: `BankAccount`

---

## Exercise 10 (Hard)
Write a short program that demonstrates that `name`, `Name`, and `NAME` are three different variables. Assign different values and print each.

**Expected Output (example):**
```
John
Jane
Bob
```

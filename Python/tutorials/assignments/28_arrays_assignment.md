# Assignment: array Module

Practice using the array module for numeric arrays.

---

## Exercise 1 (Easy)
Create an array of integers [1, 2, 3, 4, 5] using array.array('i', ...). Print it.

---

## Exercise 2 (Easy)
Create an array of floats [1.0, 2.0, 3.0]. Append 4.0. Print the array.

---

## Exercise 3 (Easy)
Create array('i', [1,2,3,4,5]). Print the first and last elements.

---

## Exercise 4 (Medium)
Convert a list [10, 20, 30] to an array. Use extend to add [40, 50]. Print.

---

## Exercise 5 (Medium)
Create array, use index to change element at index 2. Use count() to count occurrences of a value.

---

## Exercise 6 (Medium)
Create array from list, then convert back to list. Compare memory or behavior.

---

## Exercise 7 (Hard)
Create array('d', [1.5, 2.5, 3.5]). Use tolist() to convert to list. Print both.

---

## Exercise 8 (Hard)
Create two integer arrays, concatenate them (use + or extend). Print result.

---

## Exercise 9 (Hard)
Use array buffer_info() to get address and length. Print them.

---

## Exercise 10 (Hard)
Create array, use pop() to remove last element, use remove() to remove first occurrence of a value.

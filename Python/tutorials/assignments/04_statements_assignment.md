# Assignment: Python Statements

Practice writing and ordering statements.

---

## Exercise 1 (Easy)
Write three print statements that execute in order: "First", "Second", "Third".

**Expected Output:**
```
First
Second
Third
```

---

## Exercise 2 (Easy)
What happens if you put two statements on one line without a semicolon? Write the incorrect code and the error you get.

**Answer:** `SyntaxError: invalid syntax`

---

## Exercise 3 (Easy)
Write a program with 4 statements: assign `a=5`, assign `b=10`, assign `sum=a+b`, print sum.

**Expected Output:**
```
15
```

---

## Exercise 4 (Medium)
Write a program that uses semicolons to put three print statements on one line. (Demonstrate it works, but note it's not recommended.)

**Expected Output:**
```
A
B
C
```

---

## Exercise 5 (Medium)
Write 5 statements that: set `name`, set `age`, set `city`, then print them in a sentence like "Name is X, age Y, from Z".

**Example Output:**
```
Alice is 25 years old, from New York
```

---

## Exercise 6 (Hard)
Write a program with at least 6 statements that calculates and prints the area of a rectangle (length × width). Use variables for length and width.

**Example Output (length=5, width=3):**
```
Area of rectangle: 15
```

# Assignment: Python Getting Started

Practice installing and running Python.

---

## Exercise 1 (Easy)
Check if Python is installed on your system. Run the appropriate command and note the version.

**Command:** `python --version` or `python3 --version`

---

## Exercise 2 (Easy)
Create a file named `my_first_program.py` that prints "My first Python program!".

Run it from the command line and verify the output.

**Expected Output:**
```
My first Python program!
```

---

## Exercise 3 (Easy)
Enter Python interactive mode, print "Hello" and "World" on separate lines, then exit.

**Commands:**
```
python
>>> print("Hello")
>>> print("World")
>>> exit()
```

---

## Exercise 4 (Medium)
Create a file that prints three lines: your name, today's date (as text), and a greeting. Run it.

**Example Output:**
```
John
March 10, 2025
Have a great day!
```

---

## Exercise 5 (Medium)
Navigate to a different folder, create a Python file there, and run it. Document the commands you used.

---

## Exercise 6 (Hard)
If you get `'python' is not recognized`, research and list two solutions for Windows.

**Answers:**
1. Add Python to PATH during installation
2. Use `py` instead of `python` (Windows Python Launcher)

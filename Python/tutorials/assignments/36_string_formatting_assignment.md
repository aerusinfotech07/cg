# Assignment: String Formatting

Practice f-strings, format(), and format specifiers.

---

## Exercise 1 (Easy)
Use f-string to print "Hello, Alice!" with name="Alice".

---

## Exercise 2 (Easy)
Use f-string to print "5 + 3 = 8" using variables a=5, b=3.

---

## Exercise 3 (Easy)
Format 3.14159 to 2 decimal places using f-string.

---

## Exercise 4 (Medium)
Use format() to print "Name: Bob, Age: 30" with positional arguments.

---

## Exercise 5 (Medium)
Format 42 as 5-digit zero-padded: 00042. Use f-string.

---

## Exercise 6 (Medium)
Print a table: left-align "Name" (width 10), right-align "Score" (width 5). Use format specifiers.

---

## Exercise 7 (Medium)
Use f-string with expression: print the square of 7 inside the string.

---

## Exercise 8 (Hard)
Format 1000000 with comma separator: 1,000,000. Use f-string.

---

## Exercise 9 (Hard)
Create a receipt: Item $19.99, Tax 8.5%, Total. Format all with 2 decimals.

---

## Exercise 10 (Hard)
Use format() with named placeholders: "Hello {name}, you have {count} messages."

---

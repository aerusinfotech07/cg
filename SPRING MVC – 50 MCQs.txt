---

### 1. What is the front controller in Spring MVC?

A) Controller
B) DispatcherServlet
C) ViewResolver
D) HandlerAdapter

**Answer:** B

**Explanation:**
DispatcherServlet acts as the Front Controller. It receives all incoming HTTP requests and coordinates request handling.

---

### 2. Spring MVC follows which design pattern?

A) Singleton
B) MVC
C) DAO
D) Factory

**Answer:** B

**Explanation:**
Spring MVC follows the Model-View-Controller pattern.

---

### 3. Which component maps requests to controllers?

A) ViewResolver
B) HandlerMapping
C) Model
D) BeanFactory

**Answer:** B

**Explanation:**
HandlerMapping maps incoming request URLs to appropriate controller methods.

---

### 4. Which annotation marks a class as a controller?

A) @Component
B) @Controller
C) @Service
D) @Bean

**Answer:** B

**Explanation:**
@Controller defines a Spring MVC controller class.

---

### 5. DispatcherServlet is configured in:

A) web.xml
B) application.properties
C) pom.xml
D) DAO

**Answer:** A

**Explanation:**
In traditional Spring MVC, DispatcherServlet is configured in web.xml.

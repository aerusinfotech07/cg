## How to Use

1. Read **[HOME.md](HOME.md)**, then **[TUTORIAL_INDEX.md](TUTORIAL_INDEX.md)**.
2. Follow **01 → 08**; **01_Introduction** includes extended sub-notes under `notes/` and `images/`.
3. Run or review examples in `01_Introduction/examples/` where provided.
4. **Module 1** diagrams are **SVG images** in `01_Introduction/images/`.
